Exercise 1

Results in "Uncaught ReferenceError: carBrand is not defined". The variabele is only declared within it's function, so block scoped, and therefore not accessible outside of this function.


Exercise 2

Without running the function, it will result in "Uncaught ReferenceError: carBrand is not defined", since just "carBrand;" is 'seen'.


Exercise 3

It is considered bad form, "polluting the root scope", but if you do run the function like this, without declaring the value either inside or outside the function, javascript will create the variable in the root of the file.


Exercise 4

Calling up the printCar function will leak the carBrand to the root. By entering also a parameter for the function, the inner function is triggered. This inner function returns the console.log "Volvo v40". The typer variable itself stays undefined outside the functions scope.
