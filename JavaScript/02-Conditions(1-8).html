!= comparison is TRUE if value A is NOT equal to B. This comparison looks at the actual value of different datatypes
!== comparison is TRUE if value or datatype A is NOT equal to that of B.
console.log('8 != "5"? ' + (8 != "5"));
console.log('5 != "5"? ' + (5 != "5"));
console.log('5 !== "5"? ' + (5 !== "5"));

 The strict equality comparison is the reverse of the "!==" comparison. "===" will return a boolean; 
true if the compared values are exactly the same, false if they differ in value or type.
console.log('1===1? ' + (1 === 1));
console.log('1==="1"? ' + (1 === "1"));
console.log(`"hello"==='hello'? ` + ("hello" === 'hello'));
